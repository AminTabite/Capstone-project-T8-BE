# Capstone project T8

Link della repository front-end https://github.com/AminTabite/FECapstoneProject

Il capstone project e' un sito che implementa un database sui personaggi della saga Picchiaduro "TEKKEN 8" 
L'utente puo' scegliere le mosse del personaggio specifico e salvarsi le mosse da studiare,
l'Admin puo' gestire gli utenti, modificare i loro dati e volendo cancellarli.

queste sono le features disponibili:

Registrazione e Login utente con autenticazione JWT.

Ruoli USER e ADMIN con accessi differenziati alle pagine.

Home con roster dei personaggi di Tekken 8 e movelist per ogni personaggio.

Paginazione delle mosse e possibilità di aggiungere le mosse ai “Moves to Lab”.

Pagina “Moves to Lab” dove l’utente vede e gestisce le sue mosse preferite.

Pagina “My profile” per visualizzare, modificare o eliminare il proprio account.

Backoffice accessibile solo agli admin per visualizzare tutti gli utenti, modificarli o eliminarli.
